Guitar Chords Test

Go to project Go to code

Description

The purpose of this application is to assist guitarists in enhancing their musical ear through an evaluation of guitar chord identification. Within the platform, users can engage in training sessions designed to strengthen their ability to recognize guitar chords. Furthermore, they have the option to take a test in which they can customize the number of desired questions, as well as the variety and quantity of chord types they wish to identify.

During the test, a guitar chord will be randomly generated, and the goal is for the user to select the correct option corresponding to that chord. If the response is accurate, an animation will be displayed to confirm that the user chose the correct answer. In the event that the user chooses an incorrect answer, an animation will be provided to indicate the correct chord. Upon completion of the test, the user will be awarded a final score, assessing their performance in chord identification.
